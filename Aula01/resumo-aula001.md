Resumo da Aula 01 - Lógica de Programação
O que é algoritmo
Algoritmo é uma sequência de passos usados para resolver um problema ou realizar uma tarefa.

Entrada, processamento e saída;
Entrada: são os dados que recebemos.
Processamento: é o que fazemos com os dados.
Saída: é o resultado final.

Exemplo do dia a dia - Fazer café
1 Colocar água na chaleira
2 Ferver a água
3 Colocar café no filtro
4 Despejar água quente
5 Servir o café

Palavras-chave
Algoritmo: sequência de passos para resolver um problema  
Entrada: dados recebidos  
Processamento: manipulação dos dados  
Saída: resultado final
